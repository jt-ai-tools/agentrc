# 支援 (Support)

## 如何提交 Issue 並獲取協助

此專案使用 [GitHub Issues](https://github.com/microsoft/agentrc/issues) 來追蹤 Bug 和功能請求。在提交新 Issue 之前，請搜尋現有的 Issue 以避免重複。若要提交新問題，請將您的 Bug 或功能請求作為一個新的 Issue 提交。

有關使用此專案的協助和問題，請開啟 [GitHub Discussion](https://github.com/microsoft/agentrc/discussions) 或提交一個 Issue。

## Microsoft 支援政策

對此專案的支援僅限於上述列出的資源。
